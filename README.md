First- and second-year sea-ice salinity, temperature, and density from the coring sites during the Contrasts expedition in July-September 2025

Sea-ice thickness, draft, salinity, temperature, and ice density were measured during near-weekly surveys at the coring site during the CONTRASTS expedition, with four sampling events conducted at three different stations. Ice cores were extracted using a 9-cm (Mark II) internal-diameter ice corer (Kovacs Enterprise, USA). This dataset includes data from 12 coring site visits conducted between 10 July 2025 and 26 August 2025.

During each coring event, ice temperature was measured in situ from a separate temperature core using Testo 720 thermometers inserted into drill holes with a length equal to half of the core diameter, at 5-cm vertical resolution. Bulk practical salinity samples were sampled from melted core segments at 5-cm resolution from the salinity and density cores. Salinity cores were sectioned right after core extraction. Bulk practical salinity was measured using a WTW Cond 3110 conductivity meter. Ice density was determined using the hydrostatic weighing method in air and kerosene (Pustogvar and Kulyakhtin, 2016) from a separate density core processed in the freezer laboratory onboard R/V Polarstern at temperatures between –17°C and –24°C. Relative volumes of brine and gas were estimated from ice salinity, temperature, and density using Cox and Weeks (1983, doi:10.3189/S0022143000008364) for cold ice and Leppäranta and Manninen (1988, hdl.handle.net/1834/23905) for ice warmer than –2°C.

The density dataset contains the event label (1), date/time (2), geographic coordinates (3–4), core number (8), station number (9), station visit (10), ice age (11), and the presence of a melt pond above the core (12) for each coring event. For each density core, the following data are provided: manually measured ice thickness (5), ice draft (6), and core length (7). For each core section, the top (13) and bottom (14) depths measured in situ are included. Each core section also includes practical salinity (15), laboratory air temperature (16), in situ temperature (17), ice density at laboratory (18) and in situ (19) temperatures, brine volume fraction (20), and gas volume fraction at laboratory (21) and in situ (22) temperatures. Temperature values were interpolated to the depths of the density measurements. Geographic coordinates of the coring sites were measured directly. Brine volume fraction (20) is reported only for values between 0% and 40%. Ice temperatures were assumed not to exceed –0.1°C. Each core section also includes comments (23) describing specific characteristics of the sample.

The temperature dataset contains the event label (1), date/time (2), geographic coordinates (3–4), core number (8), station number (9), station visit (10), ice age (11), and the presence of a melt pond above the core (12) for each coring event. For each temperature core, the following data are provided: manually measured ice thickness (5), ice draft (6), and core length (7). For each core section, the depths (13) measured in situ are included. Each core section includes in situ temperature (14). Ice temperatures were assumed not to exceed –0.1°C.

The salinity dataset contains the event label (1), date/time (2), geographic coordinates (3–4), core number (8), station number (9), station visit (10), ice age (11), and the presence of a melt pond above the core (12) for each coring event. For each temperature core, the following data are provided: manually measured ice thickness (5), ice draft (6), and core length (7). For each core section, the top (13) and bottom (14) depths measured in situ are included. Each core section includes practical salinity (14).
